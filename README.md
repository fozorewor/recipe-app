# Recipe App 🍲

## Overview
This is a simple React-based Recipe App built using Visual Studio Code and GitHub Copilot. The app displays a list of recipes, allows users to view detailed information about each recipe, and includes a search feature to filter recipes.

---

## Features
- Display a list of recipes
- View detailed recipe information
- Search recipes by name
- Clean and simple UI using React components

---

## Tech Stack
- React (Frontend)
- JavaScript (ES6)
- CSS
- GitHub Copilot (Code generation)
